# Co-Workspace Landing Page

A simple and responsive landing page for a co-working space website built with **Vite**, **React**, and **Tailwind CSS**.

## Tech Stack
- Vite
- React
- Tailwind CSS

## Features
- Clean and modern UI
- Responsive design
- Fast development setup
- Utility-first styling with Tailwind CSS

## Getting Started

### Installation
```bash
git clone https://github.com/your-username/coworkspace-landing-page.git
cd coworkspace-landing-page
npm install
```

### Run Locally
```bash
npm run dev
```
